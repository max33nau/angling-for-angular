What Is AngularJS

In simple terms, it is a web application framework that follows the MVC design patter. For those of you who don't know MVC stands for Model-View-Controller. AngularJS is a 100% JavaScript and is compatible with both your desktop and mobile browsers. Developed by Google in 2009.

  • Model: Responsible for maintaining the application data, a model in AngularJS can be a range of data types in JavaScript such as a string, number, boolean, or just a plain javascript object.
  • View: Is pretty self explanatory, it is what the user sees. AngularJS uses HTML as its templating language and even allows you to create your own elements by using directives.
  • Controller: It is where all the magic happens, it is the place where we put our applications logic. Our model lives within our controllers in AngularJS. The controller is what controls the Model and the View based on the User interactions.($scope)

Framework Vs. Library

    Framework
  • Controls your application
  • Normally calls on your code to perform certain tasks based on the code that you have written.
  • Gives your code structure
  • You are not in charge but rather plug in your code
  • Can sometimes encapsulate useful libraries
  • Examples: ReactJS, Ember, Backbone and AngularJS
    Library
  • Packages of code where your application's code calls on the packages to perform certain tasks.
  • Acts as a toolkit
  • You are in charge
  • jQuery is a very popular library that works a lot with DOM manipulation
